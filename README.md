# e2e-project

This project runs e2e test for a login functionality 

## Libraries used:

- WebdriverIO

## You need:

- **Node.js v15.5.1**

- **Python v3**


## Instructions

- `git clone https://github.com/mfloresrnca/e2e-project.git` 
- `cd e2e-project`
- `npm install` to install the dependencies
- `npx wdio wdio.conf.js` to run test
